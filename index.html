City Latitude Project Summary:

Does the weather change the closer you get to the equator? Let's use data to find out! A Python script was written to visualize the weather of over 500 cities across the world with varying distances from the equator. This dataset was assembled using an OpenWeatherMap API call. For the list of specific cities used in this study, click Data on the navigation menu.

Once the data was gathered, Matplotlib, a Python library, was used to plot max temperature, humidity, cloudiness and wind speed vs the city's latitude. For more information, click on the Visualizations or Comparison on the navigation menu.
